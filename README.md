# CoinPy
CoinPy is an open-source project for making cryptocurrency available for everyone.

## Installation
```
pip install coinpy
```

## Examples
```
import coinpy as cp
# Fetch and Store all coins provided in Coinbase
cp.Collector().fetch_and_save()
```
